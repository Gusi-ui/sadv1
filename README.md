# sadv1 Monorepo

- **admin-panel:** Panel administrativo (Next.js)
- **mobile-app:** App móvil (React Native)
- **api:** Backend (NestJS)
- **packages:** Código compartido (UI, utils, types)

## Comandos útiles

- `npm run lint` — Linting de todo el monorepo
- `npm run format` — Formateo de todo el monorepo
